# Winter Day Calculator Backlink Starter Kit

This file is here to make outreach easier for a non-technical founder.

## Goal

Get a small number of relevant, honest backlinks from places that already talk about:

- snow days
- school closures and delays
- winter safety for families
- district communication
- local weather planning

## The easiest outreach flow

1. Pick one good page to share.
2. Use one short email template.
3. Ask for a mention only if the page is genuinely useful to their readers.

## Best target categories

- parenting blogs
- local weather blogs
- school communication blogs
- school safety resource pages
- local community resource pages
- neighborhood newsletters
- education support organizations

## Rule

Never ask for a backlink to the homepage only.
Match the request to the most relevant page.

Examples:

- family audience -> `/guides/preparing-for-snow-days`
- school operations audience -> `/guides/how-schools-decide-snow-days`
- winter hazard audience -> `/guides/snow-vs-ice-school-closures`
- morning planning audience -> `/guides/what-parents-should-check-before-6-am`

## Weekly target

- 10 outreach emails per week
- 2 to 3 follow-ups max
- 1 useful relationship is better than 50 spam emails
